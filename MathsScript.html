Maths Script

Click here for repo link.

This is a C++ script that I have made, which includes a variety of useful maths functions, that I find myself using in many of my projects. Having clamped and unclamped versions of all my Lerp-derived functions has often been useful. Included in the repository is a script called RuntimeTester which I use to measure how fast my functions are. This has been an invaluable tool for helping me know if optimisations are working, and to what extent.
