L-system OpenGL

Click here for repo link.

In second year I took an optional module focused on graphics. For it I developed this application which generates, constructs, and displays semi-random L-systems out of 3D cylinders. This project taught me lots about the graphics pipeline. Although I struggled with the complexities of vertex knitting and rendering in openGL, graphics and shaders are an area of programming that I want to improve at.
In order to generate a random L-tree I randomised the directions of rotations in the rule. I did this by setting the value of a 7 bit bitset to a randomised integer between 0 and 127. I liked this system because it would be easy to adapt to different levels of rule complexity.
